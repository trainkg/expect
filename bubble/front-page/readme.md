# 前端

## 添加依赖
例子：添加element-ui支持

```properties
"dependencies": {
"element-ui": "^2.13.1"
}
```
## 安装依赖
```sh
npm install
```

## 启动服务
```shell
npm run serve
```
## vue.config.js
这个是配合@vue/cli-service使用的配置文件。

参考文档: [Vue](https://cli.vuejs.org/zh/config/)

## babel.config.js
Babel 是一个 JavaScript 编译器

参考文档: [babel](https://www.babeljs.cn/docs/7.2.0/configuration)
